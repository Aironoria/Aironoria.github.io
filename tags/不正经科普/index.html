显示器

CRT显示器

Cathode Ray Tube：阴极射线管

曾经是应用最广泛的显示器

发光原理

  • 色显像管屏幕上的每一个像素点都由红、绿、蓝三种荧光粉组合而成
  • 通电后灯丝发热，阴极被激发，发射出三束电子束
  • 电子束经过加速、偏转后轰击各自的荧光粉
  • 荧光粉被激发，根据电子束的强弱分别发出强弱不同的红、绿、蓝三种光，混合后产生各种颜色

（高中物理orz）

LCD

Liqud Crystal Display：液晶显示器

液晶：介于固态和液态中间的状态

现在主流的显示器

发光原理

白光透射过滤光片后变成RGB三种颜色，通过调节RGB三种颜色的配比最终生成各种颜色

  • 背光源：发白光
  • TFT基板：
  • 液晶层：通过电场控制液晶层的旋转 -> 改变液晶的方向 -> 改变液晶折射的光的方向
  • 彩色滤光片：将白光转化为RGB三种颜色的光
  • 偏光片：筛选固定方向的光，以此来调节RGB的比例

LED显示器

Light-Emitting Diode：发光二极管

LED只是光源器件，所以LED显示器其实是以LED以背光源的LCD显示器，LED显示器也只是厂商的简略叫法。现在大多LCD显示器都是以LED显示器

OLED

Organic LED

现在手机主要趋势是采用OLED屏幕，如Samsung旗舰，IPhone X

似乎并没有搜到OLED显示器，可能是因为OLED成本太高的缘故。

发光原理

区别于LCD使用滤光片来改变白光颜色，OLED直接用自发光的RGB三种颜色的小灯泡，通过调节三种光的比例来发出不同颜色的光

LCD 和 OLED 的对比

  • LCD比OLED更厚，因为发光原理的关系OLED少了许多部件

  • OLED 比 LCD 更易于弯曲

    • 因为液晶层和背光层的存在，LCD无法大幅度弯，只能支持一般曲面显示器的弯曲程度。所以折叠手机、曲面屏都采用OLED

  • OLED 的黑色比 LCD 更黑

    • LCD的背光源是整块的，所以LCD显示黑色时，是通过上下偏光片相差90°来实现的，但就算两个偏光片垂直也不能保证100%的不透过率，所以不是纯黑
    • 而OLED的每个像素点都是独立的，所以呈现黑色是通过关掉某些像素点来实现的，可以完全不发光
  • 调节亮度方式不同
    • LCD直接调节背光层的电压就能控制亮度
    • OLED改变电压时会导致一些负面影响，所以只能使用PWM（Pulse Width Modulation， 脉宽调制）调节亮度。即屏幕只有开和关两种模式，在一个周期内保持50%的亮屏时间和50%的熄灭时间即达到了50%亮度的效果（附视频)
  • OLED 寿命短于 LCD
    • LCD 为无机材料，老化速度慢于OLED的有机材料
    • OLED所有像素点不一定同时点亮，导致每个像素点工作时间不同，若一个像素经常显示蓝色，那么他的蓝色就会比其他像素点衰减快，最终导致屏幕老化不均匀
